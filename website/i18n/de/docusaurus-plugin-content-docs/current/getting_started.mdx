Die Installation von [Riverpod] und das schreiben einer "Hello World" Anwendung.

## Welches Package ist das Richtige?

Zuallererst ist es wichtig zu wissen, dass [Riverpod] auf mehrere Packages verteilt ist, die alle leicht verschiedene Nutzen haben.
Die Variante von [Riverpod], die Sie installieren sollten hängt von der App ab, die sie entwickeln.

Sie können sich an der folgenden Tabelle orientieren, um zu entscheiden welches Package das Richtige für Sie ist:

| App Typ                   | Package itemName                                                                   | Beschreibung                                                                  |
| ------------------------- | ---------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| Flutter + [flutter_hooks] | [hooks_riverpod]                                                                   | Ermöglicht die gleichzeitige Nutzung von [flutter_hooks] und [Riverpod].      |
| Flutter                   | [flutter_riverpod]                                                                 | Eine Einfache Art, [Riverpod] für Flutter Apps zu nutzen                      |
| Nur Dart (ohne Flutter)   | [riverpod](https://github.com/rrousselGit/river_pod/tree/master/packages/riverpod) | Eine Version von [Riverpod] ohne alle Flutter-spezifischen Klassen.           |

## Installation

Nachdem Sie sich entschieden haben welches Package Sie installieren möchten, fügen Sie folgendes zu Ihrer `pubspec.yaml` hinzu:

<Tabs
  groupId="riverpod"
  defaultValue="hooks_riverpod"
  values={[
    { label: 'Flutter + flutter_hooks', value: 'hooks_riverpod', },
    { label: 'Flutter', value: 'flutter_riverpod', },
    { label: 'Nur Dart', value: 'riverpod', },
  ]}
>
<TabItem value="hooks_riverpod">

```yaml title="pubspec.yaml"
environment:
  sdk: ">=2.12.0 <3.0.0"
  flutter: ">=2.0.0"

dependencies:
  flutter:
    sdk: flutter
  flutter_hooks: ^0.18.0
  hooks_riverpod: ^2.0.0-dev.0
```

Dann `flutter pub get` ausführen.

</TabItem>
<TabItem value="flutter_riverpod">

```yaml title="pubspec.yaml"
environment:
  sdk: ">=2.12.0 <3.0.0"
  flutter: ">=2.0.0"

dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.0.0-dev.0
```

Dann `flutter pub get` ausführen.

</TabItem>
<TabItem value="riverpod">

```yaml title="pubspec.yaml"
environment:
  sdk: ">=2.12.0-0 <3.0.0"

dependencies:
  riverpod: ^2.0.0-dev.0
```

Dann `pub get` ausführen.

</TabItem>
</Tabs>

Das war's. Sie haben [Riverpod] zu Ihrer App hinzugefügt!
