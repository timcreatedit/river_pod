[ProviderObserver] hört Veränderungen eines ProviderContainers zu.

To use it, extend the class ProviderObserver and override the method you like to use.
Zur Verwendung können Sie einfach von der Klasse ProviderObserver erben und die Methode überschreiben, die Sie benötigen.

[ProviderObserver] hat drei Methoden:

- `didAddProvider` wird jedes mal gerufen, wenn ein Provider erzeugt wurde und seinen Wert zur bereitgestellt hat.
- `didDisposeProvider` wird jedes mal gerufen, wenn ein Provider entsorged (disposed) wird.
- `didUpdateProvider` wird jedes mal gerufen, wenn ein Provider eine Benachrichtigung versendet.

### Verwendung :

Ein einfacher Usecase für [ProviderObserver] ist es, Veränderungen in Providern zu loggen indem die Methode `didUpdateProvider` überschrieben wird.
<CodeBlock>{trimSnippet(logger)}</CodeBlock>

Jetzt wird jede Veränderung des Wertes der Provider vom Logger geloggt:

```
I/flutter (16783): {
I/flutter (16783):   "provider": "counter",
I/flutter (16783):   "newValue": "1"
I/flutter (16783): }
```
